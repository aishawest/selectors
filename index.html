This page demonstrates how CSS selectors work.

id

ids can be applied to only one item on a page

Here's how we hook an id attribute to an element:

id="id-div"

Here's how the CSS is applied to the element:

div#id-div{
    background-color:yellow;
}

Inside my-div
class

classes can be applied to more than one item on a page

Here's how we hook a class attribute to an element:

class="my-list"

Here's how the CSS is applied to the element:

li.my-list{
    background-color:pink;
}

compound

A compound selector allows us to use multiple selectors by commas. In this way we can apply the same rules to several selectors.

Here's how the CSS is applied to the element:

div#new-div,li.new-list{
    background-color:blue;
}

Here's what it looks like:

I'm inside a div with an id of new-div

child

the child selector allows us access to the elements that immediately inside the current element

Here's how the CSS is applied to the element:

ul.my-kids > li {
    background-color:blue;
}

descendant

Using the descendant selector we can access all elements at any depths within the current element

Here's how the CSS is applied to the element:

div.grandkids li {
    background-color:red;
}

mulitple classes

we can apply multiple classes by placing them in the class attribute with a space between each

Here's how we hook multiple classes attribute to an element:

class="add-border add-background"

Here's how the CSS is applied to the element:

            border:1px solid #000;
            }
